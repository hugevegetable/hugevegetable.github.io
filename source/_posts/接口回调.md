#### 1、回调 

​	调用方式分为同步调用、异步调用和回调。回调，A调用B，B执行完调用A。





```java
public class Test {
    public static void main(String[] args) {
        Passenger passenger=new Passenger();
        Driver driver=new Driver();
        passenger.askMoney(driver);
    }
}


interface Callback{
    void respond(int prize);
}
//实现接口
class Passenger implements Callback{
    void askMoney(Driver driver){
        System.out.println("多少?");
// passenger 等待 driver 实现
        driver.answer(this);
    }
    @Override
    public void respond(int prize) {
        if (prize <= 50){
            System.out.println("上车,走");
        }
        else{
            System.out.println("溜了溜了");
        }
    }
}
//回调对象
class Driver{
    void answer(Callback callback){
        System.out.println("起步价100");
        callback.respond(100);
    }
}
```
